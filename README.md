# Wireframe - Página de Aulas de Surf

## Sobre o projeto

Este projeto consiste na criação de um wireframe de média fidelidade para uma página única (one page) com o objetivo de apresentar um serviço de aulas de surf.

A proposta foi desenvolver uma interface simples, clara e funcional, aplicando conceitos de usabilidade e psicologia do design.

---

## Objetivo

O objetivo da página é permitir que o usuário compreenda rapidamente o serviço oferecido e realize o agendamento de uma aula de forma prática.

---

## Público-alvo

Pessoas interessadas em aprender surf, principalmente iniciantes e turistas que buscam uma experiência segura, guiada e acessível.

---

## Estrutura da página

A página foi organizada em seções para facilitar a navegação e compreensão:

- **Header** → identificação e navegação
- **Hero** → proposta de valor com destaque para o serviço
- **Agendamento** → incentivo direto à ação principal
- **Benefícios** → apresentação dos diferenciais do serviço
- **Footer** → informações finais

---

## Decisões de UX

- Uso de título claro e direto no hero para comunicar valor rapidamente
- CTA (Call to Action) visível e consistente ao longo da página
- Organização em blocos para facilitar a leitura e escaneabilidade
- Redução da carga cognitiva com textos curtos e objetivos
- Uso de cards para apresentar benefícios de forma estruturada

---

## Heurísticas de Usabilidade Aplicadas

### Nielsen

- **Visibilidade do status do sistema** → botões claros e ações evidentes
- **Consistência e padrões** → repetição de elementos visuais
- **Reconhecimento em vez de memorização** → informações diretas e bem organizadas

### Bastien e Scapin

- **Condução** → hierarquia visual que guia o usuário
- **Carga de trabalho** → divisão de conteúdo em blocos
- **Clareza** → linguagem simples e objetiva
